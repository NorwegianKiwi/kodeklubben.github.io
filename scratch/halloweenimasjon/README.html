Om oppgaven

I denne oppgaven skal elevene lære forskjellige måter å lage animasjoner på, og hvordan de kan sette animasjonene sammen til en lang, sammenhengende animasjon.

Oppgaven passer til:

Fag: Engelsk, Kunst og håndtverk, Programmering

Trinn: 2.-10. trinn

Tema: Halloween, Animasjon

Tidsbruk: Dobbelttime
