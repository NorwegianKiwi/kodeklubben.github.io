Om oppgaven

Denne interaktive leken viser fyrverkeri som eksploderer og smeller på himmelen når man klikker. Dette prosjektet viser et eksempel på hvordan man kan kombinere bilder og lyd til en effektfull animasjon.

Oppgaven passer til:

Fag: Kunst og håndtverk, matematikk

Anbefalte trinn: 3.-10. trinn

Tema: Koordinatsystem, bevegelser.

Tidsbruk: Dobbelttime eller mer.
