Om oppgaven

I denne oppgaven skal elevene lære å bygge fraktaler ved bruk av skilpaddekunst fra tidligere oppgaver. Denne oppgaven burde sees i sammenheng med Skilpadder hele veien.

Oppgaven passer til:

Fag: Matematikk, Kunst og håndtverk, programmering

Anbefalte trinn: 7.-10. trinn

Tema: Fraktaler, geometriske mønster, løkker, tester, rekursjon

Tidsbruk: Dobbelttime
