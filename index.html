Kodeklubbens oppgaver

Hei! På denne siden finner du mange oppgaver som du kan gjøre for å lære deg programmering. Som du ser, finnes det mange forskjellige språk. For nybegynnere anbefaler vi å starte med Scratch, men dersom du går på ungdomsskolen eller videregående så vil det også gå fint å starte med Python eller Processing.

Vi holder for tiden på å lage nye oppgave-sider for Lær Kidsa Koding, som vil bli lansert om kort tid. For å teste så kan du gå til oppgaver.kidsakoder.no/beta.

På de nye sidene finner du blant annet:

  • filter med språk, tema, fag og klassetrinn
  • fremdriftslinje i oppgavene du jobber med
  • elev- og lærermodus

Har du tilbakemeldinger til de nye sidene? Ta kontakt!
