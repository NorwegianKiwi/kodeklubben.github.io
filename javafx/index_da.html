Java er et objektorientert programmeringsspråk, og dette kurset krever forkunnskaper og erfaring med tekstbasert programmering fra tidligere. For eksempel ved å gjennomføre Kodeklubbens kurs i Python og/eller Processing. Java FX er et sett av biblioteker til Java som gjør det enklere å lage grafikk og teste programmer.
