Videregående Python

Python er et programmeringsspråk som ble utviklet tidlig på 90-tallet av Nedelenderen Guido van Rossum. I dag utvikles språket av mange i et fellesskap hvor alle som vil kan delta.

Dette kurset er beregnet på de som har programmert Python før. Dersom du ikke har programmert Python før, bør du ta en kikk på introkurset i Python.
