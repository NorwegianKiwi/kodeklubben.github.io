Om oppgaven

Denne oppgaven skal elevene jobbe videre med skrift, størrelser og farger. Elevene skal utforske forskjellige skrifttyper, se på bakgrunnsfarger og sette CSS-stil på bilder.

Denne oppgaven bygger på forrige oppgave, CSS: Style nettsider, så hvis elevene ikke har gjort den, så anbefales det å gjøre den først. Evt så kan elevene kopiere koden som ligger nederst på siden til CSS: Style nettsider.

Oppgaven passer til:

Fag: Programmering, Teknologi i praksis, Informasjonsteknologi 1

Anbefalte trinn: 8. trinn - VG2

Tema: HTML, CSS, web, fonter, klasser, id

Tidsbruk: Enkelttime
