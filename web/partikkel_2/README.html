Om oppgaven

Denne oppgaven bygger på koden elevene skrev i oppgaven Partikkel-animasjon. Så dersom elevene ikke har gjort den, så anbefaler vi å gjøre Partikkel-animasjon før elevene fortsetter på denne oppgaven.

Oppgaven passer til:

Fag: Programmering, Matematikk, Kunst og håndverk

Anbefalte trinn: 7.-10. trinn

Tema: Web, JavaScript, variabler, løkker, koordinater, animasjon

Tidsbruk: Dobbeltime eller mer.
